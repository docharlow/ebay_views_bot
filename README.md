# EBAY VIEWS BOT


A basic script that add views to you products on ebay, to rank higher.


# Features!

  - it reads product list from a given file.
  - interactive



### Installation

```bash
$ git clone https://github.com/nizarbamida/ebay_views_bot.git
$ cd ebay_views_bot
$ python3 main.py
```
### Usage:
    follow the given instruction.
